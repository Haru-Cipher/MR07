
### Lemma 4.5 核心幾何與物理意涵：完美相消干涉

Lemma 4.5 旨在證明：於給定晶格上依離散高斯分佈 (Discrete Gaussian Distribution) 抽取樣本 $\mathbf{x}$ 時，其在特定向量方向 $\mathbf{v}$ 上之複數相位的期望值將漸近於零。

就幾何與物理直覺而言，此現象可視為複數單位圓上發生的「**完美相消干涉**」(Destructive Interference)，亦即系統內多方力矩達到完美對稱與平衡的狀態。

### 達成理想期望值邊界之兩大必備條件

要使總期望值（紅色箭頭）之上界收斂至極小值（即 $\le \frac{1+\epsilon}{1-\epsilon} \cdot 2^{-n}$），系統必須同時滿足以下兩項條件，缺一不可：

#### 條件一：相位均勻分佈（取決於目標向量 $\mathbf{v}$）

* **公式條件**：$\text{dist}(\mathbf{v}, \Lambda^*) \ge \sqrt{n}/s$
*(目標向量必須與對偶晶格保持足夠距離)*
* **動態與物理意義**：向量 $\mathbf{v}$ 決定了相位的發散程度。
* **未滿足條件**：若 $\mathbf{v}$ 過於靠近整數或對偶晶格，各點之相位將趨近於整數，導致向量集中於單位圓實軸正向 $(1, 0)$。此時將產生建構性干涉（Constructive Interference），作用力集中於單側，導致期望值向量極長。
* **滿足條件**：相位將均勻散佈於單位圓的各個幅角，確保系統具備全方位的對稱向量，為互相抵消創造基礎條件。

#### 條件二：權重對稱性（取決於平滑參數 $s$）

* **公式條件**：$s \ge \eta_\epsilon(\Lambda)$
*(高斯分佈之標準差必須大於或等於晶格之平滑參數)*
* **動態與物理意義**：參數 $s$ 決定了機率分佈的平滑度與寬度。
* **未滿足條件 (過小)**：機率將極度集中於原點附近，導致單一向量（如 $\mathbf{x}=0$）之權重佔比過大。此時合力將被單一龐大權重主導，破壞整體平衡。
* **滿足條件**：高斯分佈具備足夠的「平滑性」，確保單位圓上任一方向之向量，皆存在一與之完全反向且機率權重極度相近的向量予以抗衡。
