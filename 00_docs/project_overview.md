# MR07 Project Overview

This repository contains MATLAB experiments and visualizations related to MR07, including Lemma 4.2 and Lemma 4.5 animations.

## Folder Layout

- `00_docs/`: project notes, experiment logs, and formula notes.
- `01_src/`: core MATLAB source files.
- `01_src/functions/`: reusable core functions.
- `01_src/utils/`: helper utilities.
- `02_experiments/`: experiment runner scripts.
- `03_results/`: generated experiment outputs grouped by experiment.
- `04_assets/`: images and GIFs used by README or GitHub pages.

## Current Organization

The original MATLAB scripts were moved into `01_src/` for later manual classification. README display GIFs were moved into `04_assets/gifs/`.
